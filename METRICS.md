# Metrics

## Core Metrics

### 1. Accuracy Score
Measures how consistent outputs are across runs.

### 2. Latency
Time taken per audit execution.

### 3. Token Usage
Average tokens consumed per request.

### 4. Cost per Request
Financial efficiency of each execution.

## Product Metrics
- Active users
- Requests per day
- Retention rate

## Engineering Metrics
- Error rate
- API failure rate
- Prompt deviation rate

## Target Benchmarks
- <2s latency
- <1% error rate
- Stable structured output ≥95%
