# Economics

## Cost Structure
- LLM API calls (primary cost)
- Hosting (low, serverless preferred)
- Logging/storage (minimal)

## Unit Economics
Cost per audit = API token usage + overhead

## Revenue Model
- Free tier with limited requests
- Paid tier based on usage volume
- Enterprise pricing for teams

## Optimization Strategies
- Response caching
- Prompt compression
- Model selection fallback (cheap → expensive)

## Key Metrics
- Cost per request
- Margin per user
- Token efficiency
