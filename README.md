# AI Spend Audit

AI-powered SaaS cost optimization tool that analyzes tools, detects overspending, and generates savings recommendations with automated email reports.

## Features

- AI-powered audit engine
- SaaS cost optimization recommendations
- Email report generation
- Lead capture system
- Audit history storage (Supabase)

## Tech Stack

- Next.js 15+
- TypeScript
- Supabase
- Resend Email API
- Gemini / OpenRouter AI
- TailwindCSS

## Project Structure

- /lib → core logic (AI, audit engine, integrations)
- /app/api → backend routes
- /components → UI components
- /app/results → results page
